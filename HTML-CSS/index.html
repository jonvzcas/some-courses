Meta Etiquetas

Estas etiquetas se incluyen dentro del head.

Se usa para el icono de la página web en el navegador del escritorio:

<link rel="icon" href="img/favicon-circle.png">

Se usa para asignar un icono a la página web cuando se guarda en la pantalla principal del dispositivo movil:

<link rel="apple-touch-icon" href="img/favicon-circle.png">

Cambia el color de la barra de direcciones en el dispositivo movil:

<meta name="theme-color" content="#ff6600">
