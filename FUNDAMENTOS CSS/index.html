Selectores Avanzados

Selectores de hijos

Selector de hijos directos

Solamente aplica a los elementos en el primer nivel. Se aplica el pico parentesis

  • Primavera
  • Verano
  • Otoño
  • Invierno

Selector de hijos descendientes

Va a estar representado por un espacio en blanco.

  • Primavera
  • Verano
  • Otoño
  • Invierno

Selectores de hermanos

Aplica el selector solamente a los elementos que sean hermanos (mismo nivel) y que esten por debajo del elemento de referencia. Se aplica con el caracter ~

Selector de hermanos (en general)

  • Primavera
  • Verano
  • Otoño
  • Invierno

Selector de hermanos adyacentes

Va a aplicar los estilos al elemento que coincida con la segunda parte del selector y que este inmediatamente debajo del selector de referencia. Se aplica con el simbolo de la suma +

  • Primavera
  • Verano
  • Otoño
  • Invierno

Selectores de atributo

Se pone entre corchetes el atributo al que se va a hacer referencia.

  • Inicio
  • Cursos
  • Blog
  • CV
  • Ahora
  • Notas
  • Temario CSS

Selector Universal *

  • Primavera
  • Verano
  • Otoño
  • Invierno

Prefijos de los Navegadores

  • Navegadores de Microsoft -ms-
  • Firefox -moz-
  • Todos los demás navegadores -webkit-

Autoprefixer CSS online

Autoprefixer es un PostCSS plugin que adapta el CSS a navegadores a través de prefijos.

Clic aquí para ir al código de Autoprefixer CSS online.

Ejemplo:

        /*
* Prefixed by https://autoprefixer.github.io
* PostCSS: v8.4.14,
* Autoprefixer: v10.4.7
* Browsers: last 4 version
*/

.example {
    display: -ms-grid;
    display: grid;
    -webkit-transition: all .5s;
    -o-transition: all .5s;
    transition: all .5s;
    -webkit-user-select: none;
       -moz-user-select: none;
        -ms-user-select: none;
            user-select: none;
    background: -webkit-gradient(linear, left top, left bottom, from(white), to(black));
    background: -o-linear-gradient(top, white, black);
    background: linear-gradient(to bottom, white, black);
}
